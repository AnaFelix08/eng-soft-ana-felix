# Platform Independent Model (PIM)

## Introdução

Após a validação dos requisitos junto ao cliente Sr. Geraldo, foi desenvolvido o Platform Independent Model (PIM) do Sistema de Gestão Hoteleira Bela Vista.

O PIM representa a estrutura e o comportamento do sistema de forma independente de linguagem de programação, banco de dados ou tecnologia específica.

Este documento apresenta os principais modelos UML utilizados para representar a lógica do sistema, servindo como base para as etapas futuras de desenvolvimento.

---

## Objetivo

Representar a estrutura e o comportamento do Sistema de Gestão Hoteleira Bela Vista utilizando modelos UML independentes de plataforma.

---

## Casos de uso contemplados

- Realizar Login
- Cadastrar Hóspede
- Buscar Hóspede
- Criar Reserva
- Realizar Check-in
- Registrar Consumo
- Realizar Check-out
- Atualizar Status de Limpeza
- Gerar Relatório Financeiro
- Configurar Tipos de Quartos

## Conclusão

O Platform Independent Model desenvolvido representa a estrutura lógica do Sistema de Gestão Hoteleira Bela Vista de forma independente de tecnologia.

Os diagramas de Classes, Sequência e Estados permitem compreender a organização do sistema, as interações entre seus elementos e o comportamento dos quartos durante o ciclo de hospedagem, servindo como base para a implementação futura do software.
