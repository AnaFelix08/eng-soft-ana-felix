# Semana 09 - Platform Independent Model (PIM)

## Objetivo

Desenvolver o Platform Independent Model (PIM) do Sistema de Gestão Hoteleira Bela Vista com base nos requisitos previamente validados junto ao cliente.

## Integrantes

- Ana Clara
- Ana Carolina

## Conteúdo da atividade

Esta atividade contempla a modelagem lógica do sistema utilizando UML.

Os artefatos desenvolvidos foram:

- Diagrama de Classes
- Diagrama de Sequência
- Diagrama de Estados

Todos os diagramas foram elaborados utilizando Mermaid.
