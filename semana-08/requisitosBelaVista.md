Resumo do Sistema do Hotel Bela Vista - (versão simples para validação)
--

Este documento é um resumo simples do que o sistema do hotel vai fazer.
A ideia é substituir o caderno por um sistema fácil de usar, que funcione no computador e no celular.
Quero que veja com calma e me diga:
se está tudo certo
se faltou alguma coisa
se tem algo que o senhor não precisa

QUARTOS
--
O sistema vai mostrar todos os quartos em uma tela, usando cores:
- Verde - Quarto livre
- Vermelho - Quarto ocupado
- Amarelo - Quarto sujo (precisa limpar)
- Não deixar colocar duas pessoas no mesmo quarto ao mesmo tempo
- Permitir cadastrar tipos de quarto (ex: simples, luxo, suíte)
- Permitir mudar o preço dos quartos quando precisar
- Marcar quando o quarto está sujo ou limpo

**Exemplo do dia a dia:**

- Quando o hóspede sair → quarto fica "sujo"
- Depois que limpar → quarto volta para "livre"

HÓSPEDES E RESERVAS
--
O sistema vai permitir:
- Cadastrar hóspede (nome, CPF, telefone e data de nascimento)
- Procurar hóspede pelo nome ou CPF
- Fazer reservas com data de entrada e saída
- Evitar erro de reservar quarto já ocupado

**Exemplo:**

Se o Tião tentar colocar alguém em um quarto que já está ocupado, o sistema vai avisar.


CONSUMOS (frigobar e serviços)
--
Vai ser possível:
- Registrar consumo do quarto (ex: água, refrigerante, cerveja, serviço)

O sistema vai guardar automaticamente:
- data
- hora
- quem registrou

No final, o sistema soma o  valor das diárias e o valor dos consumos sozinho e mostra o valor total para cobrar do hóspede.

RELATÓRIO DO DINHEIRO
--
O sistema vai mostrar quanto entrou de dinheiro em um período (ex: dia, semana ou mês). Separando os valores das diárias e os valores dos consumos

ONDE O SISTEMA VAI FUNCIONAR
--
O sistema vai funcionar tanto no computador quanto no celular, sem precisar instalar nenhum programa. A ideia é que ele seja simples de usar e rápido no dia a dia.

SEGURANÇA
--
Cada pessoa terá um acesso diferente, evitando bagunça e erro:
- Seu Geraldo → vê tudo
- Recepcionista → faz check-in e reservas
- Camareira → marca limpeza
- Funcionário → registra consumo

O QUE NÃO VAI TER NESSE PRIMEIRO MOMENTO
--
Para manter tudo mais simples no começo, algumas coisas não vão entrar agora, como integração com Booking ou Airbnb, emissão automática de nota fiscal, controle de folha de pagamento dos funcionários e aplicativo separado para celular, já que o sistema vai funcionar direto pelo navegador. Se no futuro o senhor precisar de alguma dessas funções, elas podem ser adicionadas depois.
